Why Discord?

Why Discord? Their are many reasons why discord is a great tool for learning programming. Number 1, its a messaging platform, so you can use it for other purposes as well. Number 2, it supports sending code that is properly formatted, making it very easy to read. Number 3, its very user friendly, and it doesnt take long to learn the user interface. But most importantly, they have great support for making bots that users can interact with.

Discord Bots

The best thing about discord is the amazing support for bots. Bots are a great way to add fun features to your server, and just as fun to make. There is alot to understand before that though, and I HIGHLY recommend that you have at least a basic understanding of the language you want to use, because programming a discord bot can become complicated very quickly. So, lets unpack the image to the right.
1 - user's name, often used to check permissions.
2 - the bot's prefix. this is really important because this tells the bot "hey this might be a command", because it checks every message that is sent.
3 - the actual command. in this example it is 'help', which is a command every bot should have.
4 - not as important as some of the other parts of the image, this is the bot's profile picture. Just kind of a fun thing to customize.
5 - bot's name, again, not as useful as some of the other parts of the image, but it can be useful in rare instances.
6 - the bot account indicator. This is really important, because it indicates that this account is a bot. Most bots use this to check if a message was sent by a bot, and if it was, dont respond to it.
7 -last but not least, the bots response. This is what the bot sends, and based on what command it recieves, it sends different things, it can even send a styled up embed

Code Snippits

Sending code can be a pain. Luckily Discord has a great feature that formats the code and makes it more readable. First, you need to find the back tick on your keyboard, (right under the escape key), because it is used to signify that you are sending code. There are 3 ways to send a code snippit. Number 1, using 1 back tick, like this: `code here`. This should only be used if your code fits on one line. Number 2, using 3 back ticks, like this: ```code``` this shouldnt realy be used, but it works. and finally, number 3, using 3 back ticks and specifying the language, then hitting enter like the example on the left. This is the best way, and should be used most of the time.
