Modbot

Purpose

The goal of Modbot is to provide a free, user friendly bot that is versitile, and can assist you in moderating and controlling your server. Below you will find all the information you need to get started using the bot.

Commands

Modbot is meant for, well, moderation. The Commands below are all of the base commands the bot has. It also has the ability to use custom commands you write! If that is too daunting, check out the list of commands other people have made and see if one fits your need.

List:

  • ban
  • kick
  • mute
  • server info
  • bot info
  • user info
  • reaction roles
  • auto roles
  • ban words
  • custom commands

Docs

The docs explain a little bit more about custom commands. They also have example commands, and explain the usages of each variable more in-depth.
