Discord and Bots Official inDev Bot

The inDev Discord Bot

I started working on the inDev Discord bot during the corona virus pandemic. It is useful, easy to use, and has an interesting sense of humor. While I dont recommend trying out bot developement if you are new to programming, if you have at least intermediate knowledge, bot developement is a great way to futher your understanding of whatever language or languages you choose.

Features

  • Play
  • Pause
  • Stop
  • Queue - view queue
  • clear
  • Reaction Roles
  • Add Nickname when given a role
  • Add role when user joins
  • meme
  • DBmeme
  • add meme
  • Bot info
  • Server info
  • User info
  • Mute
  • Kick
  • Ban
  • Ban word
  • clear messages
  • set prefix

Developed using JDA
